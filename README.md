# LookLook 

目前包括三个模块：知乎日报、网易头条、每日看看，其中每日看看数据来自干活集中营。项目旨在打造一款极简的阅读app，优化用户体验。未来会加入更多有意思的模块。（项目还存在很多问题，持续更新中，欢迎star）
